# 📃 1) About Project
"독서인을 위한 종합 커뮤니티 사이트"

독서의 활성화를 위한 다양한 커뮤니티 기능을 고려한 책을 위한, 책에 의한, 독서 커뮤니티 사이트

팀이름	마지막 잎새
프로젝트 기간	2022.12.30 ~ 2023.01.25
팀장	강매란
팀원	김광진, 김세헌, 김휘수, 박승근, 박유림
개발목표	훈련과정에서 배운 것을 기반으로 하여 CRUD 기능 구현에 초점을 둔다.
팀원 각 자의 개성을 살리면서 책 커뮤니티 사이트에서 필요한 기능을 맡아 구현한다.
